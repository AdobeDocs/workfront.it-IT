Puoi visualizzare tutti gli oggetti disponibili tramite l’API Adobe Workfront, insieme ai relativi campi, in API Explorer. Per visualizzare l’API Explorer, fai clic [qui](https://developer.adobe.com/workfront/api-explorer/).

Per informazioni più dettagliate sull’utilizzo di API Explorer, consulta [Utilizzo di API Explorer](../../wf-api/general/using-api-explorer.md).
