# Panoramica di Adobe Cloud Drive

Adobe Cloud Drive è un’applicazione desktop che consente di montare i progetti di archiviazione cloud di Adobe come un’unità nel computer Mac o Windows. I progetti Workfront vengono visualizzati come cartelle e puoi aprire, creare, modificare e salvare i file direttamente nei progetti Workfront nell’archiviazione cloud di Adobe, utilizzando qualsiasi applicazione desktop.

I file in Adobe Cloud Drive sono gli stessi che vengono visualizzati in Workfront e Frame.io, pertanto le modifiche apportate sul desktop sono disponibili nei progetti Workfront connessi in entrambi i prodotti.

[Visualizza una panoramica video delle funzionalità disponibili in Adobe Cloud Drive.](https://video.tv.adobe.com/v/3492492){target="_blank"}

Per ulteriori informazioni sull&#39;archiviazione a cui Adobe Cloud Drive si connette, consulta [Panoramica sull&#39;archiviazione cloud di Adobe](/help/quicksilver/review-and-approve-work/esm-overview.md).

## Operazioni possibili con Adobe Cloud Drive

* Accedi ai progetti dall’archiviazione cloud di Adobe come cartelle nel Finder (Mac) o Esplora file (Windows).
* Aprire, creare e modificare file in applicazioni Adobe Creative Cloud, Microsoft Office o in qualsiasi applicazione installata nel computer.
* Salva le modifiche localmente e consenti ad Adobe Cloud Drive di sincronizzarle con i progetti nell’archiviazione cloud di Adobe.
* Contrassegnare i file per l&#39;accesso offline quando si è in viaggio o si lavora senza una connessione di rete. Quando la connessione viene ripristinata, gli aggiornamenti vengono sincronizzati con il file nel progetto e puoi quindi rimuovere i file dalla cache locale per liberare spazio su disco.
* Collaborare con i membri del team negli stessi file di progetto.

## Chi può utilizzare Adobe Cloud Drive

Per utilizzare Adobe Cloud Drive, la tua organizzazione deve trovarsi nel pacchetto Workflow Ultimate, con l’archiviazione cloud Adobe abilitata.

Nell’unità vengono visualizzati solo i progetti che utilizzano Adobe Cloud Storage. I progetti Workfront legacy non vengono visualizzati. Per ulteriori informazioni sulla differenza, consulta [Differenze tra l&#39;archiviazione cloud Adobe e l&#39;archiviazione Workfront legacy](/help/quicksilver/review-and-approve-work/esm-overview.md#differences-between-adobe-cloud-storage-and-legacy-workfront-storage).

Se Adobe Cloud Drive non è disponibile per il download o se l&#39;organizzazione gestisce le installazioni dell&#39;applicazione, contatta l&#39;amministratore Workfront.
