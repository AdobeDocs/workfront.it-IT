# Installare Adobe Cloud Drive

Per utilizzare i progetti di archiviazione cloud Adobe sul desktop, installare Adobe Cloud Drive sul computer Mac o Windows.

Per informazioni su Adobe Cloud Drive e sulle operazioni possibili con esso, consulta [Panoramica di Adobe Cloud Drive](/help/quicksilver/documents/adobe-cloud-drive/adobe-cloud-drive-overview.md).

## Prerequisiti

Prima di installare Adobe Cloud Drive, verifica quanto segue:

* Il dispositivo soddisfa i requisiti di sistema elencati nella sezione [Requisiti di sistema](#system-requirements) in questo articolo.
* La tua organizzazione utilizza una versione di Workfront che supporta l’archiviazione cloud di Adobe.

Se la tua organizzazione gestisce le installazioni dell’applicazione e non trovi Adobe Cloud Drive disponibile per il download, contatta il tuo amministratore Workfront.

## Requisiti di sistema

**Sistemi operativi:**

* macOS 13.5 o versione successiva (Silicio Intel e Apple)
* Windows 11 22H2 (22621.6060) o successivo (x64)
* Windows 11 23H2 (22631.6199) o successivo (ARM)

**Hardware:**

* RAM minima: 4 GB
* 500 MB di spazio libero su disco per l&#39;applicazione
* Spazio aggiuntivo per i file memorizzati in cache (varia in base all’utilizzo)

**Rete:**

* Connessione Internet per la configurazione iniziale e la sincronizzazione
* Connessione stabile consigliata per le prestazioni

Per assicurarsi che Adobe Cloud Drive possa raggiungere i servizi da cui dipende, consenti gli endpoint di rete elencati in [endpoint di rete Adobe](https://helpx.adobe.com/in/enterprise/kb/network-endpoints.html).

## Scarica il programma di installazione

1. Vai a [Distribuzione software Adobe](https://experience.adobe.com/#/downloads).

1. Accedi con il tuo Adobe ID.
1. Nell&#39;elenco dei download **General**, individuare **Adobe Cloud Drive**.
1. Selezionare il programma di installazione del sistema operativo:

   * **macOS:** Adobe Cloud Drive per Mac (ARM64 o Intel)
   * **Windows:** Adobe Cloud Drive for Windows (x64 o ARM64)

1. Fai clic su **Scarica**.

   >[!IMPORTANT]
   >
   >Se il programma di installazione di Adobe Cloud Drive non è visibile, è possibile che la tua organizzazione non disponga di una versione di Workfront che supporta l’archiviazione cloud di Adobe. Contatta l’amministratore di Workfront.

## Installare Adobe Cloud Drive

Utilizzare la procedura per il sistema operativo in uso.

### Installa su macOS

1. Apri il file `.dmg` scaricato.
1. Fare doppio clic sull&#39;icona di installazione visualizzata.
1. Seguire la procedura guidata di configurazione su schermo.

1. Quando richiesto, accedi con il tuo Adobe ID.

   Se è già in esecuzione un&#39;altra applicazione desktop Adobe e hai effettuato l&#39;accesso, Adobe Cloud Drive utilizza automaticamente tale sessione attiva.

### Installa su Windows

1. Eseguire il programma di installazione di `.exe` scaricato.

1. Seguire le istruzioni della procedura guidata di installazione.
1. Fare clic su **Installa** quando richiesto.
1. Avviare **Adobe Cloud Drive** dal menu Start.
1. Quando richiesto, accedi con il tuo Adobe ID.

## Verificare l&#39;installazione

Dopo l&#39;installazione, Adobe Cloud Drive viene visualizzato come un&#39;unità montata sul computer:

* **macOS:** nella barra laterale del Finder in **Percorsi**, come **Adobe Cloud Drive**.
* **Windows:** In Esplora file in **Questo PC**, come **Adobe Cloud Drive**.

Dovresti visualizzare i progetti di archiviazione cloud Adobe come cartelle all’interno dell’unità.

Per informazioni sull&#39;utilizzo dei progetti nell&#39;unità, vedere [Utilizzare Adobe Cloud Drive](/help/quicksilver/documents/adobe-cloud-drive/use-adobe-cloud-drive.md).
