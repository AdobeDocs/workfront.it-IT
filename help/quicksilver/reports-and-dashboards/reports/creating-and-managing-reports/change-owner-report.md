# Cambiare il proprietario di un rapporto

<!-- Audited: 11/2024 -->

Non è possibile modificare il proprietario di un report. Tuttavia, quando si copia un report, si diventa automaticamente il proprietario del report copiato.

Per ulteriori informazioni che sono state spostate in altri articoli, consulta:

* [Crea una copia di un report](../../../reports-and-dashboards/reports/creating-and-managing-reports/create-copy-report.md): questo articolo spiega come creare una copia di un report per diventare il proprietario di un report.
* [Eseguire e recapitare un report con i diritti di accesso di un altro utente](../../../reports-and-dashboards/reports/creating-and-managing-reports/run-deliver-report-access-rights-another-user.md): in questo articolo viene illustrato come aggiornare il campo **Eseguire il report con i diritti di accesso di:** di un report, che deve includere solo un utente attivo.
* [Creare un report sulle attività di reporting](../../../reports-and-dashboards/reports/report-usage/create-report-reporting-activities.md): questo articolo spiega come trovare informazioni sui report esistenti, ad esempio chi possiede un report, con chi è condiviso un report, quale utente è elencato nel campo **Eseguire il report con i diritti di accesso di:** e così via.
