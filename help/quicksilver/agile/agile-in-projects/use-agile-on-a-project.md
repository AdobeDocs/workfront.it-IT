# Utilizzare Agile in un progetto

Puoi sfruttare le funzionalità Agile per il tuo progetto senza le sfide amministrative che in genere accompagnano le pratiche Agile (come la gestione del backlog di un team, la creazione di iterazioni e così via). Per ulteriori informazioni, consulta [Gestire un progetto nella [!UICONTROL Visualizzazione Agile]](../../manage-work/projects/manage-projects/manage-projects-in-agile-view.md).
