# Attività del primo trimestre del 2026 per le funzioni dell’agenzia Adobe Workfront

Questo articolo descrive le funzioni di Workfront specifiche per l’agenzia che vengono rilasciate durante il primo trimestre del 2026.

<!--keep the sentence below for all future quarterly release pages-->

Per un elenco di tutte le funzionalità delle agenzie rilasciate per Adobe Workfront, consulta [Attività di rilascio delle agenzie Adobe Workfront: indice articolo](/help/quicksilver/product-announcements/product-releases/agencies-release-activity/agencies-release-activity-article-index.md).

## Regolare le percentuali di mansioni nelle schede delle tariffe

>[!NOTE]
>
>Data di rilascio in produzione: xx novembre 2025

Ora puoi applicare modifiche basate sulla percentuale a una o più tariffe di fatturazione dei ruoli all’interno di una scheda tariffa. Invece di aggiornare manualmente ogni tariffa una alla volta, puoi applicare le modifiche a tutte le tariffe selezionate in una singola operazione.
