# Attività di rilascio di Workfront Fusion: settimana 10 maggio 2021

Questa pagina descrive tutti i miglioramenti effettuati in Adobe Workfront Fusion la settimana del 10 maggio 2021.

Per un elenco di tutte le modifiche recenti, vedi [Attività di rilascio di Adobe Workfront Fusion](../../../product-announcements/product-releases/fusion-release-activity/fusion-release-activity.md).

Per un elenco delle correzioni di bug recenti in Workfront Fusion, vedi [Aggiornamenti alla manutenzione di Workfront](https://one.workfront.com/s/article/Workfront-Maintenance-Updates-1882317350) e controlla eventuali aggiornamenti etichettati Workfront Fusion Maintenance Update.

## Scegli la versione API di Workfront nel modulo Workfront > Chiamata API personalizzata

Al fine di aumentare la flessibilità nell’accesso all’API Workfront, abbiamo reso possibile scegliere la versione dell’API Workfront che si desidera utilizzare nel modulo Workfront > Chiamata API personalizzata. Ciò ti consente di utilizzare funzionalità API disponibili in precedenza che potrebbero comunque essere pertinenti alle esigenze della tua organizzazione.

Per ulteriori informazioni, consulta [Moduli Adobe Workfront](../../../workfront-fusion/apps-and-their-modules/workfront-modules.md).
