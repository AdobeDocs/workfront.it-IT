### API versione 10 disponibile in anteprima {#api-version-10-now-available-in-preview}

La versione 10 dell’API è ora disponibile nell’ambiente di anteprima. Sarà disponibile nell’ambiente di produzione con la versione 18.3. Per ulteriori informazioni, consulta [Novità dell’API versione 10](../../../../wf-api/api/new-api-version-10.md). 

### API versione 3 non più supportata {#api-version-3-no-longer-supported}

Il supporto per la versione 3 dell’API terminerà a novembre 2018. Per ulteriori informazioni sulla pianificazione del supporto per le versioni dell’API Workfront, vedi [Controllo delle versioni e pianificazione del supporto API](../../../../wf-api/api/api-version-support-schedule.md).

### Fine del supporto TLS {#tls-support-is-ending}

Per garantire una sicurezza ottimale, Workfront richiede l’aggiornamento di tutte le integrazioni e il traffico web che utilizzano TLS 1.0 o versioni precedenti per utilizzare TLS 1.1 o versioni successive. Questa modifica è già attiva nell’ambiente Anteprima.

Per ulteriori informazioni, consulta  [TLS 1.2 da richiedere in Adobe Workfront](../../../../product-announcements/announcements/announcement-archive/tls-1-disabled.md).

### Fine del supporto JSONP {#jsonp-support-is-ending}

Poiché JSONP (Javascript con spaziatura interna) è un vecchio standard con vulnerabilità di sicurezza note, Workfront non supporterà più JSONP a partire da novembre 2018. Per ulteriori informazioni, consulta [Terminazione del supporto per JSONP](../../../../wf-api/api/ending-support-jsonp.md).

### Rimozione applicazione Flash {#flash-application-removal}

Tutte le funzionalità basate sulla tecnologia di Flash sono pianificate per la rimozione da Workfront con la versione 2018.3 e sostituite con una nuova soluzione.

Per informazioni dettagliate sugli strumenti di Flash legacy e sulle relative sostituzioni, consulta [Sostituzione di strumenti basati su Flash in Adobe Workfront](../../../../product-announcements/announcements/announcement-archive/replace-flash-tools.md).

### Varie opzioni di Single Sign-On da rimuovere {#various-single-sign-on-options-to-be-removed}

Le seguenti opzioni Single Sign-On (SSO) sono ora supportate solo in una capacità limitata; saranno rimossi dal prodotto nella seconda metà del 2019:

* LDAP

   Per ulteriori informazioni sull&#39;utilizzo di LDAP con Workfront, vedi [Configurare Adobe Workfront con LDAP](../../../../administration-and-setup/manage-workfront/security/removal-various-sso-options.md).

* Active Directory

   Per ulteriori informazioni sull&#39;utilizzo di Active Directory con Workfront, vedere [Configurare Adobe Workfront con Active Directory](../../../../administration-and-setup/manage-workfront/security/removal-various-sso-options.md).

* SAML 1.1

   Per ulteriori informazioni sull&#39;utilizzo di SAML 1.1 con Workfront, vedi [Configurare Adobe Workfront con SAML 1.1](../../../../administration-and-setup/manage-workfront/security/removal-various-sso-options.md).

Di seguito è riportata la sequenza temporale di rimozione per queste opzioni SSO:

* **Supporto limitato:** Agosto 2018 - Gennaio 2019

   Verranno risolti i problemi principali; non verranno affrontati i problemi di scarsa gravità.

* **Obsolescenza:** Gennaio 2019 - Agosto 2019

   Non supportato.

* **Non più disponibile in Workfront:** Agosto 2019

Queste modifiche sono in preparazione all’esperienza di autenticazione avanzata, attualmente disponibile solo per un numero limitato di clienti.

### Aggiornamento legacy di Workfront Fusion - 20 settembre 2018 {#legacy-workfront-fusionupdate-september-20-2018}

La legacy Workfront Fusion è una piattaforma di integrazione che consente di collegare Workfront a più applicazioni di terze parti.

Con l&#39;ultimo aggiornamento, Legacy Workfront Fusion fornisce ora un design migliorato che rende più intuitivo il processo di creazione di nuovi FLO. Le modifiche specifiche includono:

* **Nuovo FLO:** Facendo clic sul nuovo FLO gli utenti passano ora alla schermata Designer anziché alla prima richiesta di selezione del connettore. Questa modifica ti aiuta ad orientarti verso l&#39;FLO complessivo e capire che qualsiasi FLO dovrebbe iniziare con un Evento.
* **Aggiungi funzione:** Il selettore delle funzioni è stato aggiornato per facilitare la ricerca e la selezione della funzione desiderata. Le descrizioni delle funzioni, insieme ai campi di input e output, sono state aggiornate per aggiungere indicazioni intuitive. La nuova sezione &quot;Più popolare&quot; consente di iniziare a utilizzare le funzioni più comunemente utilizzate.
