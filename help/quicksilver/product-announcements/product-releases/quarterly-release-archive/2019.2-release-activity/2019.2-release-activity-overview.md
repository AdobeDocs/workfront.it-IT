# Panoramica dell’attività di rilascio 2019.2

Questa pagina fornisce informazioni sulle funzionalità incluse nella versione 2019.2. Per scaricare il contenuto in formato PDF, vai [qui](https://documentation.my.workfront.com/library/a/6626eb71-16b2-4460-bd81-9adcd4953eb5).

>[!NOTE]
>
>La versione 2019.2 è stata resa disponibile nell’ambiente di produzione nelle date seguenti:
>
>* *Ambiente di produzione Workfront per cluster 2, 4 e 6 - 23 maggio*
>* *Ambiente di produzione Workfront per cluster 1, 3 e 5 - 24 maggio*
>* *Ambiente di produzione a prova di Workfront - 21 maggio*


La versione 2019.2 contiene miglioramenti sia per gli amministratori di sistema che per altri utenti.

## Miglioramenti per gli amministratori

* [Specificare i ruoli del processo nelle voci orarie](../../../../product-announcements/product-releases/quarterly-release-archive/2019.2-release-activity/2019.2-administrator-enhancements.md)

   È ora possibile specificare un ruolo di lavoro quando si registra il tempo in Workfront.

* [Consenti agli utenti senza una licenza di correzione di applicare azioni e risolvere commenti](../../../../product-announcements/product-releases/quarterly-release-archive/2019.2-release-activity/2019.2-proofing-enhancements.md)

   Gli utenti senza una licenza di correzione possono ora applicare azioni ai commenti e risolvere i commenti.

## Miglioramenti per tutti gli utenti

### Miglioramenti alla gestione delle risorse

* [Nuovo livello di accesso alla gestione delle risorse](../../../../product-announcements/product-releases/quarterly-release-archive/2019.2-release-activity/2019.2-resource-management-enhancements.md#new)

   È ora disponibile un nuovo livello di accesso per Gestione risorse, che consente agli utenti con una licenza Piano di modificare l’accesso.


* [Ordinare progetti per priorità Portfolio nel planner risorse](../../../../product-announcements/product-releases/quarterly-release-archive/2019.2-release-activity/2019.2-resource-management-enhancements.md#order)

   Ora è possibile ordinare i progetti in base alla priorità del loro portfolio nel Planner risorse.

* [Miglioramento del filtro del planner risorse](../../../../product-announcements/product-releases/quarterly-release-archive/2019.2-release-activity/2019.2-resource-management-enhancements.md#resource)

   L’esperienza di filtraggio nel Planner risorse è ora più semplice quando aggiungi un nuovo campo (Gruppo) ai criteri di filtro incorporati.

   È ora possibile abilitare un’impostazione nel Planner risorse che ti consente di visualizzare la priorità del progetto in base al portfolio a cui è assegnato.

* [Condividere i filtri nel planner risorse](../../../../product-announcements/product-releases/quarterly-release-archive/2019.2-release-activity/2019.2-resource-management-enhancements.md#share)

   È ora possibile condividere i filtri creati nel Planner risorse con utenti, team, gruppi, ruoli o aziende selezionati.

* [Tracciamento avanzamento percentuale completamento su problemi negli strumenti di Agile](../../../../product-announcements/product-releases/quarterly-release-archive/2019.2-release-activity/2019.2-agile-enhancements.md#track)

### Libreria Workfront disponibile {#workfront-library-now-available}

La libreria Workfront è un hub di contenuti digitali aziendali che consente di gestire l’intero ciclo di vita delle risorse digitali dell’organizzazione.

### API versione 3 non più supportata {#api-version-3-no-longer-supported}

Il supporto per la versione 3 dell’API è terminato a novembre 2018. Per ulteriori informazioni sulla pianificazione del supporto per le versioni dell’API Workfront, vedi [Controllo delle versioni e pianificazione del supporto API](../../../../wf-api/api/api-version-support-schedule.md).

### Fine del supporto TLS {#tls-support-is-ending}

Per garantire una sicurezza ottimale, Workfront richiede l’aggiornamento di tutte le integrazioni e il traffico web che utilizzano TLS 1.0 o versioni precedenti per utilizzare TLS 1.1 o versioni successive. Questa modifica è già attiva nell’ambiente Anteprima.

Per ulteriori informazioni, consulta [TLS 1.2 da richiedere in Adobe Workfront](../../../../product-announcements/announcements/announcement-archive/tls-1-disabled.md).

### Rimozione applicazione Flash {#flash-application-removal}

Tutte le funzionalità basate sulla tecnologia di Flash sono state rimosse da Workfront a partire dalla versione 2018.3 e sostituite con una nuova soluzione.

Per informazioni dettagliate sugli strumenti di Flash legacy e sulle relative sostituzioni, consulta [Sostituzione di strumenti basati su Flash in Adobe Workfront](../../../../product-announcements/announcements/announcement-archive/replace-flash-tools.md).

### Varie opzioni di Single Sign-On da rimuovere {#various-single-sign-on-options-to-be-removed}

Le seguenti opzioni Single Sign-On (SSO) sono ora supportate solo in una capacità limitata; saranno rimossi dal prodotto nella seconda metà del 2019:

* LDAP

   Per ulteriori informazioni sull&#39;utilizzo di LDAP con Workfront, vedi [Configurare Adobe Workfront con LDAP](../../../../administration-and-setup/manage-workfront/security/removal-various-sso-options.md).

* Active Directory

   Per ulteriori informazioni sull&#39;utilizzo di Active Directory con Workfront, vedere [Configurare Adobe Workfront con Active Directory](../../../../administration-and-setup/manage-workfront/security/removal-various-sso-options.md).

* SAML 1.1

   Per ulteriori informazioni sull&#39;utilizzo di SAML 1.1 con Workfront, vedi [Configurare Adobe Workfront con SAML 1.1](../../../../administration-and-setup/manage-workfront/security/removal-various-sso-options.md).

Di seguito è riportata la sequenza temporale di rimozione per queste opzioni SSO:

* Supporto limitato: agosto 2018 - gennaio 2019

   Saranno risolte le principali questioni presentate nel corso di tale periodo; non verranno affrontati i problemi di scarsa gravità.

* Obsolescenza: Gennaio 2019 - Agosto 2019

   Non supportato.

* Non più disponibile in Workfront: Agosto 2019
