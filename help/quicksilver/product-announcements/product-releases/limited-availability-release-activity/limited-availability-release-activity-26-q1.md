# Attività della versione del primo trimestre 2026 per le funzioni di disponibilità limitata di Adobe Workfront

Questo articolo descrive le funzioni di disponibilità limitata di Workfront rilasciate durante il primo trimestre del 2026.

<!--keep the sentence below for all future quarterly release pages-->

Per un elenco di tutte le funzionalità di disponibilità limitata rilasciate per Adobe Workfront, vedere [Attività di rilascio delle funzionalità di disponibilità limitata di Adobe Workfront: indice articolo](/help/quicksilver/product-announcements/product-releases/limited-availability-release-activity/limited-availability-release-activity-article-index.md).

## Regolare le percentuali di mansioni nelle schede delle tariffe

>[!NOTE]
>
>Data di rilascio in produzione: 20 novembre 2025

Ora puoi applicare modifiche basate sulla percentuale a una o più tariffe di fatturazione dei ruoli all’interno di una scheda tariffa. Invece di aggiornare manualmente ogni tariffa una alla volta, puoi applicare le modifiche a tutte le tariffe selezionate in una singola operazione.
