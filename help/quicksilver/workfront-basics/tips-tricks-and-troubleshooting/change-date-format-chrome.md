# Modificare il formato per le date in [!DNL Adobe Workfront]

<!--this article used to be called "Change the date format in Adobe Workfront when using Chrome". The team decieded to make it more generic and hide the steps. Also see drafted content below-->

Per modificare il formato delle date in [!DNL Adobe Workfront], quali [!UICONTROL Data completamento pianificata], [!UICONTROL Data completamento effettivo]oppure [!UICONTROL Data di completamento prevista], da GG/MM/AAAA a MM/GG/AAAA o viceversa, ad esempio, è necessario modificare le impostazioni della lingua nel browser.

Se la lingua predefinita nel browser è impostata su *[!UICONTROL Inglese (Stati Uniti)]*, le date vengono visualizzate nel formato MM/GG/AAAA .

Per modificare le impostazioni della lingua in [!DNL Chrome] o qualsiasi altro browser, è necessario modificare le impostazioni di tale browser. I passaggi per modificare le impostazioni di un browser variano da browser a browser. Fai riferimento alle [!UICONTROL Aiuto], [!UICONTROL Preferenze]oppure [!UICONTROL Impostazioni] per scoprire come modificare le impostazioni.
