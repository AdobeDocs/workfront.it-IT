# Panoramica del server Adobe Workfront MCP

Il server MCP (Model Context Protocol) [!DNL Adobe Workfront] consente di lavorare con i dati Workfront tramite una conversazione in linguaggio naturale in un assistente di intelligenza artificiale supportato. Invece di fare clic su Workfront per trovare un progetto, aggiornare un’attività o controllare un’approvazione, chiedi all’assistente AI in inglese semplice e l’assistente AI esegue il lavoro al tuo posto.

Per utilizzare il server MCP di Workfront non è necessario conoscere l’API di Workfront o altro su MCP. Configura l’assistente AI per parlare con Workfront una volta, quindi inizia a chiedere.

## Configurazione e utilizzo del server MCP di Workfront

Prima di poter utilizzare il server MCP di Workfront, un amministratore di Workfront deve abilitare l’accesso al server MCP in Workfront e collegare l’assistente AI a Workfront. Per ulteriori informazioni, vedere [Configurare il server Adobe Workfront MCP](/help/quicksilver/workfront-basics/workfront-mcp-server/configure-workfront-mcp-server.md).

Dopo la configurazione, puoi chiedere all’assistente AI di trovare, creare, aggiornare e gestire gli elementi di Workfront. Per ulteriori informazioni sulle azioni disponibili, le richieste di esempio e la gestione dei dati, vedere [Utilizzare il server Adobe Workfront MCP](/help/quicksilver/workfront-basics/workfront-mcp-server/use-workfront-mcp-server.md).
