# Panoramica del server Adobe Workfront MCP

{{highlighted-preview-article-level}}


Il server MCP [!DNL Adobe Workfront] connette l&#39;istanza di Workfront a una piattaforma AI agente come [!DNL Claude] o [!DNL ChatGPT]. Dalla piattaforma dell’agente di intelligenza artificiale, puoi trovare, creare, aggiornare e gestire gli elementi di Workfront effettuando richieste in linguaggio naturale.

Ad esempio, puoi chiedere:

* *Mostra tutti i progetti attivi per il team Marketing marchio.*
* *Aggiornare l&#39;attività &quot;Design Review&quot; in modo che termini il 18 aprile.*
* *Invia un promemoria agli approvatori della risorsa &quot;Video campagna di primavera&quot; che non hanno risposto.*

Non è necessario conoscere l’API Workfront o il funzionamento dei server MCP per utilizzare il server MCP Workfront.

>[!IMPORTANT]
>
>Attualmente, il server MCP di Workfront è disponibile solo per i clienti della regione degli Stati Uniti che utilizzano AWS.

## Che cos&#39;è un server MCP

Un server MCP è un punto di connessione che consente a una piattaforma di intelligenza artificiale di funzionare con un altro sistema. Il server MCP di Workfront è ciò a cui la piattaforma di intelligenza artificiale si connette per poter leggere e agire sui dati Workfront per tuo conto.

MCP è l&#39;acronimo di Model Context Protocol. Questo è lo standard che definisce il modo in cui le piattaforme di intelligenza artificiale e i sistemi esterni comunicano tra loro.

## Configurare la connessione

Il server MCP di Workfront funziona con qualsiasi piattaforma di intelligenza artificiale compatibile con MCP, ad esempio [!DNL Claude] o [!DNL ChatGPT]. Prima di poterla utilizzare, è necessario che si verifichi quanto segue:

* Un amministratore Workfront deve abilitare l’accesso al server MCP nell’istanza Workfront.
* L’utente (o l’amministratore) deve collegare la piattaforma di intelligenza artificiale a Workfront.

Per ulteriori informazioni, vedere [Configurare il server Adobe Workfront MCP](/help/quicksilver/workfront-basics/workfront-mcp-server/configure-workfront-mcp-server.md).

## Iniziare a utilizzare il server Workfront MCP

Dopo la configurazione, puoi chiedere alla piattaforma di agenti di intelligenza artificiale di trovare, creare, aggiornare e gestire gli elementi Workfront in linguaggio naturale.

Per ulteriori informazioni, incluse richieste di esempio, aspetti da tenere presenti e informazioni su dati e sicurezza, vedere [Utilizzare il server MCP di Adobe Workfront](/help/quicksilver/workfront-basics/workfront-mcp-server/use-workfront-mcp-server.md).
