# [!UICONTROL Markdown] moduli

In un [!DNL Adobe Workfront Fusion] puoi utilizzare la [!UICONTROL Markdown] moduli per convertire Markdown in HTML e HTML in Markdown.

Se hai bisogno di istruzioni su come creare uno scenario, vedi [Crea uno scenario in [!DNL Adobe Workfront Fusion]](../../workfront-fusion/scenarios/create-a-scenario.md).

Per informazioni sui moduli, consulta [Moduli in [!DNL Adobe Workfront Fusion]](../../workfront-fusion/modules/modules.md).

## [!UICONTROL Markdown per HTML]

Questo modulo converte Markdown in HTML.

Quando si configura questo modulo, vengono visualizzati i campi seguenti.

<table style="table-layout:auto"> 
 <col> 
 <col> 
 <tbody> 
  <tr> 
   <td role="rowheader">[!UICONTROL Markdown]</td> 
   <td> <p>Inserisci il testo normale formattato per Markdown.</p> </td> 
  </tr> 
  <tr> 
   <td role="rowheader">[!DNL GitHub Flavored Markdown] </td> 
   <td> <p>Abilita questa opzione per convertire GitHub Flavored Markdown in HTML.</p> <p>Per ulteriori informazioni, consulta il mar[!DNL ]scheda di controllo a cascata [!DNL GitHub] documentazione.</p> </td> 
  </tr> 
  <tr> 
   <td role="rowheader">[!UICONTROL Sanitize]</td> 
   <td>Selezionare un’opzione per indicare se si desidera rimuovere i tag HTML dal testo o da HTML di escape.</td> 
  </tr> 
 </tbody> 
</table>
