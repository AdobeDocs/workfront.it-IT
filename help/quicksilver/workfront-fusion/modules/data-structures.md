# Strutture dei dati in [!DNL Adobe Workfront Fusion]

Una struttura dati è un documento che descrive in dettaglio il formato dei dati trasferiti [!DNL Adobe Workfront Fusion]. In base a questo documento, l&#39;editor di scenari è in grado di capire quale modulo restituisce o riceve quale tipo di dati. I documenti della struttura dati vengono utilizzati più comunemente per serializzare/analizzare formati di dati come JSON, XML, CSV e altri.

Puoi creare una struttura dati facendo clic sul pulsante [!UICONTROL Creare una nuova struttura dati] nel [!UICONTROL Panoramica della struttura dati] nelle impostazioni del modulo che richiede la specifica della struttura dati.

I tipi di dati supportati sono descritti in [[!UICONTROL Tipi di moduli]](../../workfront-fusion/modules/module-types.md) articolo.

## Generatore della struttura dati

Non è sempre necessario creare strutture di dati. Puoi renderlo più semplice utilizzando un modello del nostro generatore integrato. Fornendo un esempio di dati, il generatore creerà automaticamente una struttura dati basata sull’esempio di dati inserito. La struttura dati creata può quindi essere modificata manualmente.
